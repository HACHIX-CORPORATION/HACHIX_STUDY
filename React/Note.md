# Reactとは
1. UIを作るためのJavaScript用ライブラリ
2. Facebook、Yahoo、ATOM、Airbnbなど、名だたる有名な企業で採用されています。

# コンセプト
1. Declarative
2. Component-Based
3. Learn Once, Write Anywhere

# 特徴
1. レンダリングが高速
2. 大規模なアプリケーションほど向いている
3. SPAのアプリケーションに向いている
